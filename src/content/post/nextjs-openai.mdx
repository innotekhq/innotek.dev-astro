## Introduction

In this post we'll focus on creating a user interface with a single text input where users can submit prompts to OpenAI. For brevity, we'll assume you've already set up Node.js and will start directly from npm installations.

## Starting with Next.js and TypeScript

First, let's set up our Next.js application with TypeScript support. Ensure you have a basic understanding of [Next.js App Router](https://nextjs.org/docs/app) before proceeding.

1. **Create a Next.js App**: In your terminal, execute the following command to create a new Next.js app:

```bash
npx create-next-app@latest nextjs-openai --ts --eslint --tailwind --app
```
Replace nextjs-openai with your preferred project name. When asked if you want to use the src/ directory, select No.
The above command automatically sets up TypeScript, ESLint and TailwindCSS for your project, including generating a tsconfig.json file. Verify its presence in your project directory.

2. **Set the OpenAI API Key**

To make calls to OpenAI's API, you'll need to set your API key as an environment variable. You can obtain your API key from the [OpenAI dashboard](https://platform.openai.com/api-keys).

Once you have this, change into your project directory:

```bash
cd nextjs-openai
```

Create a .env file with the following contents:

```
OPENAI_API_KEY=<your open ai key>
```

And finally install the OpenAI SDK:

```bash
npm i openai
```

## Implementing a REST API in Next.js

Next.js App Router allows us to easily implement API routes, which we will use to interact with OpenAI's API.

1. **Create an API Route:**

 - Create a file under `app/api/send-prompt/`, such as `route.ts`.
 - This file will export an async function named POST that handles the requests. The naming is important, as Next.js will automatically route `POST /api/send-prompt` request to this function.

2. **Implementing the API Endpoint** (`app/api/send-prompt/route.ts`):

```typescript
import { NextResponse } from "next/server";
import OpenAI from "openai";

const openai = new OpenAI({
  apiKey: process.env.OPENAI_API_KEY,
});

export async function POST(req: Request) {
  try {
    const body = await req.json();
    const prompt = body.prompt;
    const params: OpenAI.Chat.ChatCompletionCreateParams = {
      messages: [{ role: "user", content: prompt }],
      model: "gpt-3.5-turbo",
    };
    const chatCompletion: OpenAI.Chat.ChatCompletion =
      await openai.chat.completions.create(params);
    console.log(chatCompletion);
    const response = {
      response: chatCompletion?.choices[0].message.content?.trim(),
    };
    return NextResponse.json(response);
  } catch (error) {
    console.log(error);
    return NextResponse.error();
  }
}
```

## Building the UI

Let's create a simple UI that consists of a text input and a submit button to send prompts to our backend.

1. **Main Page** In app/page.tsx, create a form with a text input and a button. Use TailwindCSS for styling:

```tsx
"use client"

import { useState } from 'react';

export default function Home() {
  const [prompt, setPrompt] = useState('');
  const [response, setResponse] = useState('');
  const [isLoading, setIsLoading] = useState(false);

  const handleSubmit = async (e) => {
    e.preventDefault();
    setIsLoading(true);
    // API call will be implemented here
  };

  return (
    <div className="container mx-auto p-4">
      <form onSubmit={handleSubmit} className="flex flex-col gap-3">
        <textarea
          className="p-2 border border-gray-300 rounded text-gray-700"
          placeholder="Enter your prompt"
          value={prompt}
          onChange={(e) => setPrompt(e.target.value)}
        ></textarea>
        <button
          type="submit"
          className="bg-blue-500 text-white p-2 rounded hover:bg-blue-600"
          disabled={isLoading}
        >
          {isLoading ? "Sending..." : "Send Prompt"}
        </button>
      </form>
      {response && (
        <div className="mt-4 p-4 border border-gray-300 rounded">
          <p>{response}</p>
        </div>
      )}
    </div>
  );
}
```

This code creates a form with a textarea for the prompt and a button to submit it. It also includes a section to display the response from the backend.

2. **Implementing the API Call**

In the handleSubmit function, use the fetch API to send the prompt to your REST API endpoint and handle the response:

```tsx
  const handleSubmit = async (e) => {
    e.preventDefault();
    setIsLoading(true);
    try {
      const res = await fetch('/api/send-prompt', {
        method: 'POST',
        headers: {
          'Content-Type': 'application/json',
        },
        body: JSON.stringify({ prompt }),
      });
      if (res.ok) {
        const data = await res.json();
        setResponse(data.response);
      } else {
        throw new Error('Network response was not ok.');
      }
    } catch (error) {
      console.error('Fetch error:', error);
      setResponse('Failed to fetch response.');
    }
    setIsLoading(false);
  };
```

This function sends the user's prompt to the `/api/send-prompt`` endpoint, waits for the response, and then updates the state to display it.
